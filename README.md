# daily-planner

a sveltekit version of a simple day planner (todo list) i made [here](https://github.com/justisabelll/day-planner)

setup local db

[my sql local setup](https://orm.drizzle.team/docs/guides/mysql-local-setup)

install dependencies

```bash
bun install
```

run the dev server

```bash
bun run dev
```
